# 前端目标模式 v1

状态：已接入协议 1.16 / contract-v16

## 已建立边界

- 已装备发射器且其 `TargetSpecDto` 支持格子或实体目标时，玩家可通过 `F` 或地图下方按钮进入目标模式。
- 准星从玩家所在格开始，沿当前小键盘、Vi 或 WASD 预设的八方向键移动，并在地图边界和核心声明的 Chebyshev 射程内钳制。
- `Enter` 确认目标，`Esc` 或按钮取消。确认格存在多个实体时按稳定实例 ID 选择第一个实体；没有实体时回退为格子目标，并提交 `FireTarget`。
- 准星使用独立 HTML 交互层，跟随 PixiJS 相机偏移和缩放。目标模式、准星位置和输入提示均为前端瞬时状态，不进入存档、回放或 state hash。
- 前端只负责形成稳定选择；可见性、视线、射程、弹药消费和实际碰撞仍由 Rust 核心在命令执行时重新验证。
