# License scope

RoguelikeFansBand Rewrite separates software licensing from creative-content licensing.

## MPL-2.0

Unless a file says otherwise, original software is licensed under the Mozilla Public License 2.0. This includes:

- Rust, TypeScript, JavaScript and other source code;
- build scripts, developer tools and tests;
- protocol and data schemas;
- configuration used to build or run the software.

The complete MPL-2.0 text is available in the repository root `LICENSE` file.

## CC BY-SA 4.0

Unless a file says otherwise, original creative content is licensed under Creative Commons Attribution-ShareAlike 4.0 International. This includes:

- `README.md` and original documentation under `design/` or `docs/`;
- original gameplay data and prose under `content/`;
- original artwork, audio and other creative assets under `assets/`.

The complete license text is in `LICENSES/CC-BY-SA-4.0.txt`.

## Exclusions

Legacy RoguelikeFansBand, FrogComposband and Angband source code, data, text, binaries and assets are not included in this repository and are not relicensed by this project.

Third-party files retain their own licenses and must carry an SPDX identifier, adjacent license notice or an entry in `NOTICE`.
